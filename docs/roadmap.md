# Roadmap inicial

## Fase 0 (semana 1)
- Documentacion base (hecho).
- Esqueleto API y healthcheck (hecho).
- Definir modelo de datos y enums.

## Fase 1 (semanas 2-3)
- Implementar modelos SQLAlchemy y migraciones Alembic.
- CRUD de propiedades/personas/contratos.
- Endpoint GeoJSON para mapa.
- Autenticacion JWT con roles.

## Fase 2 (semanas 4-5)
- Motor de cobranza y pagos parciales.
- Reajustes (UF/IPC) y notificaciones de vencimiento.
- Documentos: upload, versionado, metadatos.
- Timeline de estados y auditoria.

## Fase 3 (semanas 6-7)
- Mapa interactivo completo en frontend.
- Dashboards de ocupacion y mora.
- Tickets de mantencion y SLA.

## Fase 4 (semanas 8+)
- Portal propietario (lectura y reportes).
- Integracion con Excel de legado (import/export).
- IA basica para clasificacion de documentos y resumen de contratos.
- Hardening: backups, monitoreo, CI/CD.
